## COS 3.3

### 发布时间

2016.05.13

### 版本特性

- 功能完善
  - 跨域 OPTIONS 无需签名
  - 设置 Headers 参数覆盖写入
  - 开启静态后自动匹配 /


- 新增支持的文件操作
  - 上传同名文件覆盖
  - 移动/重命名文件
  - 指定 Object 独立权限
- 体验优化
  - 错误码使用明文返回
  - 控制台多处操作优化

### 文档更新

- 同名文件覆盖：[COS 产品文档 - 开发者手册 - RESTful API - 创建文件](https://www.qcloud.com/doc/product/227/%E5%88%9B%E5%BB%BA%E6%96%87%E4%BB%B6)
- 同名文件覆盖：[COS 产品文档 - 开发者手册 - RESTful API - 移动文件](https://www.qcloud.com/doc/product/227/%E7%A7%BB%E5%8A%A8%E6%96%87%E4%BB%B6)

## COS 3.2

### 发布时间

2016.05.04

### 版本特性

- 自定义文件元数据
  - 支持四种标准 HTTP 头部
  - 支持用户自定义 x-cos-meta- 前缀的头部
- 跨域访问 CORS 支持
  - 支持上传和下载时自定义跨域
  - 支持响应 OPTIONS 跨域请求
- 官网统一鉴权

## COS 3.0

### 发布时间

2016.01.15

### 版本特性

- 与 CDN 解耦
  - 开放外网访问域名
  - 允许用户配置默认域名
  - 新增自定义域名绑定源站

  - 外网另计费、内网不计费

- 新增认证使用流程
- Referer 防盗链功能升级
  - 支持配置白名单
  - 支持配置黑名单
- 改版控制台用户体验
